📱 **Инструкция для телефона (Android/iOS)**

**Восстановление из бекапа (рекомендуется):**
1. Скачайте приложение AmneziaVPN:
   - Android: [Google Play](https://play.google.com/store/apps/details?id=org.amnezia.vpn&hl=ru)
   - iOS: [App Store](https://apps.apple.com/us/app/amneziavpn/id1600529900)
2. Нажмите "Приступить"
3. Выберите "Восстановить из резервной копии"
4. Выберите скаченный файл бекапа из бота
5. Подключитесь

**Настройка из конфигурации:**
1. Скачайте приложение AmneziaVPN (ссылки выше)
2. Нажмите "Приступить"
3. Скопируйте конфигурацию из бота
4. Вставьте и подключитесь

⚠️ Настоятельно рекомендуем настраивать выборочный обход.
Почти все Российские адреса заблокированы.
